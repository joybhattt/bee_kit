The following project aims to create a Basic Enviornment and Essentials engine (bee_engine)


The gist of the engine:

-----------------enviornment----------------------
audio : { sound_regsiter, sound_play, volume ....}
rendering : { texture_register, quad_render, ....}
windowing : { creating_window, getting_input, ....}
--------------------------------------------------

---------------------essentials-------------------
utility : { threads, timer, error ....}
physics : { object_collison, gravity ....}
networking : development of a protocol using udp to mimic QUIC protocol.
---------------------------------------------------
